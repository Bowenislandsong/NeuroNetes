NeuroNetes

Agent-Native Kubernetes Framework for LLM Workloads

Production-ready orchestration for AI agents with GPU-first scheduling, token-aware autoscaling, and conversation-level routing

View Examples Get Started

Why NeuroNetes?

Traditional Kubernetes wasn't designed for LLM-based agent workloads. NeuroNetes fills these gaps with specialized controllers, schedulers, and metrics built specifically for AI agents.

Ultra-Fast Scale-from-Zero

TTFT-aware autoscaling with built-in warm pools, snapshot/restore of model weights, and lazy container initialization for instant response times.

ğŸ®

GPU-First Scheduling

Topology-aware GPU bin-packing, MIG partition orchestration, gang scheduling for tensor/pipeline parallel jobs, and NUMA-aware placement.

Token-Aware Autoscaling

Native metrics including tokens/sec, TTFT, queue depth, and context length. Function-of-input-length scaling with session-aware concurrency management.

Session & Conversation Affinity

Sticky routing by conversation ID, graceful session handoff during scaling, and state-aware load balancing with short-lived state management.

Model Lifecycle Management

Token-Level Observability

Cost/SLA-Aware Placement

Multi-objective scheduling considering SLO, $/token, and carbon footprint. Graceful degradation to smaller models with spot instance integration.

Quick Start

Prerequisites

  • Kubernetes 1.25+
  • GPU Operator (for GPU workloads)
  • Metrics Server
  • Prometheus (for custom metrics)

Installation

# Install CRDs
kubectl apply -f config/crd/

# Deploy controllers
kubectl apply -f config/deploy/

# Verify installation
kubectl get pods -n neuronetes-system

Local Development

# Start local services (Redis, NATS, Weaviate, Prometheus, Grafana)
make docker-compose-up

# Access services:
# - Grafana: http://localhost:3000 (admin/admin)
# - Prometheus: http://localhost:9090
# - Weaviate: http://localhost:8080
# - NeuroNetes: http://localhost:8081

# Create local Kind cluster
make dev

# Deploy example resources
kubectl apply -f config/samples/
