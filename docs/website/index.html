AWS (EKS) Installation

Deploy NeuroNetes on Amazon EKS with GPU-enabled EC2 instances

1. Create EKS Cluster

# Configure variables
export CLUSTER_NAME=neuronetes-prod
export REGION=us-west-2
export K8S_VERSION=1.28

# Create cluster with GPU node groups
eksctl create cluster \
  --name $CLUSTER_NAME \
  --region $REGION \
  --version $K8S_VERSION \
  --node-type m5.xlarge \
  --nodes 2 \
  --nodes-min 2 \
  --nodes-max 5 \
  --managed \
  --with-oidc

# Add GPU node group (P4d instances with A100 GPUs)
eksctl create nodegroup \
  --cluster $CLUSTER_NAME \
  --region $REGION \
  --name gpu-nodes \
  --node-type p4d.24xlarge \
  --nodes 1 \
  --nodes-min 0 \
  --nodes-max 10 \
  --node-labels workload=ai-agent \
  --managed

2. Install GPU Operator

# Add NVIDIA Helm repository
helm repo add nvidia https://helm.ngc.nvidia.com/nvidia
helm repo update

# Install NVIDIA GPU Operator
helm install gpu-operator nvidia/gpu-operator \
  --namespace gpu-operator \
  --create-namespace \
  --set operator.defaultRuntime=containerd

3. Install NeuroNetes

# Install CRDs
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/crd/

# Deploy controllers
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/deploy/

# Verify installation
kubectl get pods -n neuronetes-system

Azure (AKS) Installation

Deploy NeuroNetes on Azure Kubernetes Service with NC-series VMs

1. Create AKS Cluster

# Configure variables
export RESOURCE_GROUP=neuronetes-prod
export LOCATION=eastus
export CLUSTER_NAME=neuronetes-aks

# Create resource group
az group create \
  --name $RESOURCE_GROUP \
  --location $LOCATION

# Create AKS cluster
az aks create \
  --resource-group $RESOURCE_GROUP \
  --name $CLUSTER_NAME \
  --node-count 2 \
  --node-vm-size Standard_D4s_v3 \
  --enable-managed-identity \
  --enable-addons monitoring \
  --kubernetes-version 1.28 \
  --generate-ssh-keys

# Add GPU node pool (NCv3 with V100 or NC A100 v4)
az aks nodepool add \
  --resource-group $RESOURCE_GROUP \
  --cluster-name $CLUSTER_NAME \
  --name gpunodes \
  --node-count 1 \
  --node-vm-size Standard_NC24ads_A100_v4 \
  --labels workload=ai-agent \
  --node-taints sku=gpu:NoSchedule

2. Install GPU Drivers

# Get cluster credentials
az aks get-credentials \
  --resource-group $RESOURCE_GROUP \
  --name $CLUSTER_NAME

# Install NVIDIA GPU Operator
helm repo add nvidia https://helm.ngc.nvidia.com/nvidia
helm repo update

helm install gpu-operator nvidia/gpu-operator \
  --namespace gpu-operator \
  --create-namespace \
  --set operator.defaultRuntime=containerd

3. Install NeuroNetes

# Install CRDs
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/crd/

# Deploy controllers
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/deploy/

# Verify installation

GCP (GKE) Installation

Deploy NeuroNetes on Google Kubernetes Engine with A100/H100 GPUs

1. Create GKE Cluster

# Configure variables
export PROJECT_ID=neuronetes-prod
export REGION=us-central1
export ZONE=us-central1-a
export CLUSTER_NAME=neuronetes-cluster

# Set project and enable APIs
gcloud config set project $PROJECT_ID
gcloud services enable container.googleapis.com
gcloud services enable compute.googleapis.com

# Create GKE cluster
gcloud container clusters create $CLUSTER_NAME \
  --region $REGION \
  --cluster-version 1.28 \
  --machine-type n1-standard-4 \
  --num-nodes 2 \
  --enable-autoscaling \
  --min-nodes 2 \
  --max-nodes 10 \
  --enable-stackdriver-kubernetes \
  --addons HorizontalPodAutoscaling,HttpLoadBalancing

# Add GPU node pool (A100)
gcloud container node-pools create gpu-pool \
  --cluster $CLUSTER_NAME \
  --region $REGION \
  --accelerator type=nvidia-tesla-a100,count=1 \
  --machine-type a2-highgpu-1g \
  --num-nodes 1 \
  --min-nodes 0 \
  --max-nodes 10 \
  --enable-autoscaling \
  --node-labels=workload=ai-agent

2. Install GPU Drivers

# Install NVIDIA GPU device drivers
kubectl apply -f https://raw.githubusercontent.com/GoogleCloudPlatform/container-engine-accelerators/master/nvidia-driver-installer/cos/daemonset-preloaded.yaml

# Verify GPU nodes
kubectl get nodes -l cloud.google.com/gke-accelerator=nvidia-tesla-a100

3. Install NeuroNetes

# Install CRDs
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/crd/

# Deploy controllers
kubectl apply -f https://raw.githubusercontent.com/Bowenislandsong/NeuroNetes/main/config/deploy/

# Verify installation
kubectl get pods -n neuronetes-system
