== Getting Started ==

1. Verify the installation:

   kubectl get pods -n {{ include "neuronetes.namespace" . }}
   kubectl get crds | grep neuronetes.io

2. Check CRDs are installed:

   kubectl get models,agentclasses,agentpools,toolbindings

3. Deploy a sample model:

   cat <<EOF | kubectl apply -f -
   apiVersion: neuronetes.io/v1alpha1
   kind: Model
   metadata:
     name: llama-3-8b
   spec:
     weightsURI: s3://models/llama-3-8b/
     size: 16Gi
     quantization: int8
     cachePolicy:
       priority: medium
       pinDuration: 1h
   EOF

4. Create an AgentClass:

   cat <<EOF | kubectl apply -f -
   apiVersion: neuronetes.io/v1alpha1
   kind: AgentClass
   metadata:
     name: my-assistant
   spec:
     modelRef:
       name: llama-3-8b
     maxContextLength: 8192
     slo:
       ttft: 500ms
       tokensPerSecond: 50
   EOF

5. Deploy an AgentPool:

   cat <<EOF | kubectl apply -f -
   apiVersion: neuronetes.io/v1alpha1
   kind: AgentPool
   metadata:
     name: my-pool
   spec:
     agentClassRef:
       name: my-assistant
     minReplicas: 1
     maxReplicas: 10
     prewarmPercent: 20
   EOF
